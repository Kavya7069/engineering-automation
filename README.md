Engineering Workflow Automation Tool
A full-stack Engineering Workflow Automation Tool that lets engineers trigger common workflows (CSV processing, file conversion, API calls, and report generation) through a simple web dashboard backed by FastAPI.

Features
FastAPI-based REST backend

Clean HTML/CSS/JS frontend dashboard

Four ready-made workflows:

CSV Data Processing

File Conversion (JSON/XML/CSV)

External API Integration

Report Generation (HTML/JSON/Markdown)

Execution history and real-time output panel

Basic error handling and clear status messages
